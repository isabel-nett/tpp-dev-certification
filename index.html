About the Certification Process
Welcome to the steps that you will work through for your successful integration of Triple Play Pay’s API.

Here is what you will accomplish:
  1. Familiarizing yourself with the API Documentation and GitHub repository
  2. Integrating the API using our test API key
  3. Submitting our required test data
  4. Receiving the certificate
